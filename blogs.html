Question-1: What are the different ways to select an element in the DOM?

There are five ways to select an element in the DOM.
  1. document.getElementById("<id>"): By using the ID of the element.
  2. document.getElementsByTagName("<tag_name>"): By using the tag name of the element (or elements). Returns an HTMLCollection of all the elements of the specified tag name.
  3. document.getElementsByClassName("<class_name>"): By using the class name of the element (or elements). Returns an HTMLCollection of all the elements of the specified class name.
  4. document.querySelector("<css_selector>"): By using the CSS selectors. Returns only the first element of the specified selector.
  5. document.querySelectorAll("<css_selector>"): By using the CSS selectors. Returns a Nodelist of all the elements of the specified selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML: Gets the HTML content (markup, elements, and text) inside an element as a string. On setting values, replaces all content inside the element with the specified HTML.
  • textContent: Gets the full text content of an element and all its descendants, including text in hidden elements and <script> or <style> tags (if present). On setting values, replaces all content (including child elements) inside the element with the specified text.
  • innerText: Gets the visible text content of an element and all its descendants. On setting values, behaves the same as textContent.

Question-3: What is event delegation in the DOM?

Instead of adding an event listener to each and every similar element, we can add an event listener to their parent element and call an event on a particular target using the .target property of the event object. This process is called event delegation.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts at the target element and propagates upward through its parent elements to the root of the DOM, triggering event handlers on those parent elements unless explicitly stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create elements:
    document.createElement("<tag_name>");
  • Add elements:
    parentElement.appendChild(childElement);
  • Remove elements:
    parentElement.removeChild(childElement);
